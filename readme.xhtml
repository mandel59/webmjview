webmjview

文字情報基盤 文字情報一覧表から漢字を検索します。

使用方法

検索欄に探したい漢字の読みや画数などを入力して検索します。空白で区切ることで、複数の条件を使って候補を絞り込めます。

全角英数字を使っても検索できます。

例

たら
訓読みが「たら」の漢字を検索
カ はな
音読みが「カ」、訓読みが「はな」の漢字を検索
ポンド
「ポンド」と読む漢字を検索（外来語はカタカナで入力してください）
=64
総画数が64画の漢字を検索
>=40
総画数が40画以上の漢字を検索
=2-4
総画数が2画以上4画以下の漢字を検索
[25]
卜部の漢字を検索（康熙字典の部首番号です）
[30]-5
口部5画の漢字を検索
[187]-0-2
馬部0画から2画までの漢字を検索
u3400-db5
UCS符号位置U+3400からU+3DB5の範囲で検索
mj2500-99
MJ002500からMJ002599までの範囲で検索
#KS00
平成明朝体グリフのグリフ名に「KS00」を含む字を検索
